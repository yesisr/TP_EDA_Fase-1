\begin{abstract}
    Este relatório descreve o desenvolvimento da primeira fase do projeto da unidade curricular de Estruturas de Dados Avançadas, que consistiu na implementação de um sistema de simulação de efeitos de antenas utilizando listas ligadas em linguagem C.\\
    O trabalho aborda a modelação do problema, a estrutura de dados desenvolvida, as funcionalidades implementadas e os resultados obtidos.\\
    A solução proposta permite carregar dados de antenas a partir de ficheiros, gerir dinamicamente a lista de antenas, calcular automaticamente zonas de efeito nefasto e apresentar os resultados de forma organizada. O projeto demonstra a aplicação prática de conceitos fundamentais de estruturas de dados dinâmicas e boas práticas de desenvolvimento de software.
    \end{abstract}

\section{Descrição do Problema}
\label{sec:problema}

O problema consiste em simular o comportamento de múltiplas antenas de transmissão distribuídas numa área geográfica representada por uma matriz bidimensional. Cada antena possui as seguintes características:

\begin{itemize}
    \item \textbf{Posição}: definida por coordenadas (x,y) na matriz
    \item \textbf{Frequência}: representada por um caractere (A-Z)
\end{itemize}

O efeito nefasto ocorre quando duas antenas com a mesma frequência estão alinhadas e uma está exatamente ao dobro da distância da outra em relação a um ponto comum. Matematicamente, para duas antenas A1(x1,y1) e A2(x2,y2) com mesma frequência, existem dois pontos de efeito nefasto:

\begin{itemize}
    \item Ponto 1: (2x1 - x2, 2y1 - y2)
    \item Ponto 2: (2x2 - x1, 2y2 - y1)
\end{itemize}

Estes pontos podem estar dentro ou fora da área da matriz considerada. Quando múltiplos pares de antenas estão presentes, todos os possíveis pontos de efeito nefasto devem ser calculados.

\section{Solução Implementada}
\label{sec:solucao}

\subsection{Estrutura de Dados}
A estrutura de dados principal foi implementada como uma lista ligada simples, conforme o seguinte código:

\begin{verbatim}
/**
 * @struct Antena
 * @brief Estrutura que representa uma antena no sistema
 */
typedef struct Antena {
    int x;                  /**< Coordenada x na matriz */
    int y;                  /**< Coordenada y na matriz */
    char frequencia;        /**< Frequência de operação (A-Z) */
    struct Antena *next;    /**< Ponteiro para próxima antena */
} Antena;

/**
 * @struct PontoEfeito
 * @brief Estrutura que representa um ponto com efeito nefasto
 */
typedef struct PontoEfeito {
    int x;                  /**< Coordenada x do ponto */
    int y;                  /**< Coordenada y do ponto */
    struct PontoEfeito *next; /**< Ponteiro para próximo ponto */
} PontoEfeito;
\end{verbatim}

\subsection{Arquitetura do Sistema}
O sistema foi organizado nos seguintes módulos principais:

\begin{itemize}
    \item \textbf{main.c}: Contém a lógica principal do programa e interface com o utilizador
    \item \textbf{antenas.h}: Estrutura dinâmica das antenas
    \item \textbf{func.c}: Contém as funções do programa
    \item \textbf{antenas.txt}: Ficheiro de texto para a matriz de antenas
\end{itemize}

\subsection{Principais Algoritmos}
\subsubsection{Carregamento de Antenas a partir de Ficheiro}
O algoritmo de carregamento segue os seguintes passos:
\begin{enumerate}
    \item Abrir ficheiro em modo leitura
    \item Ler dimensões da matriz
    \item Para cada linha subsequente:
    \begin{enumerate}
        \item Ler caracteres da linha
        \item Para cada caractere diferente de '.':
        \begin{enumerate}
            \item Criar nova antena com coordenadas (linha,coluna)
            \item Inserir na lista ligada
        \end{enumerate}
    \end{enumerate}
    \item Fechar ficheiro
\end{enumerate}

\subsubsection{Cálculo de Pontos de Efeito Nefasto}

Para cada par de antenas com mesma frequência:

\begin{enumerate}
    \item \textbf{Calculo das diferenças}:
    \begin{itemize}
        \item $\Delta x = x_2 - x_1$
        \item $\Delta y = y_2 - y_1$
    \end{itemize}
    
    \item \textbf{Pontos de efeito nefasto}:
    \begin{itemize}
        \item Ponto anterior: $(x_1 - \Delta x,\ y_1 - \Delta y)$
        \item Ponto posterior: $(x_2 + \Delta x,\ y_2 + \Delta y)$
    \end{itemize}
    
    \item \textbf{Verificação do resultado}:
    \begin{itemize}
        \item Descartar pontos que coincidam com antenas existentes
        \item Ignorar coordenadas fora dos limites da matriz
    \end{itemize}
\end{enumerate}
\newpage
\textbf{Exemplo prático}:
Para duas antenas em $(2,2)$ e $(3,3)$:
\begin{itemize}
    \item Diferenças: $\Delta x = 1$, $\Delta y = 1$
    \item Pontos calculados:
    \begin{itemize}
        \item $(1,1)$
        \item $(4,4)$
    \end{itemize}
\end{itemize}
